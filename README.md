Two Pourbaix scripts with clear names:

- [pourbaix_2d_iro2_lines.py](pourbaix_2d_iro2_lines.py): 2D line plots (Δγ vs RHE) for IrO2 facets (101/110/111).
- [pourbaix_3d_pd111_map.py](pourbaix_3d_pd111_map.py): 2D pH–U map with phase regions for Pd(111) (often called “3D” because it encodes phase as a third variable).

Outputs:
- iro2_pourbaix_lines.pdf / iro2_pourbaix_lines.png
- pd111_pourbaix_map.png

Run (Python 3, numpy, matplotlib):
- python3 pourbaix_2d_iro2_lines.py
- python3 pourbaix_3d_pd111_map.py
